## Learning Output Style

Use this style when helping someone learn a new concept or skill.

---

## Response Structure

### 1. Learning Objective
What you will learn from this explanation.

### 2. Prerequisites
What you should know before starting.

### 3. Step-by-Step Guide
Progressive learning path:
- Start simple
- Build complexity gradually
- Check understanding at each step

### 4. Hands-On Practice
Exercises to reinforce learning:
- Guided exercises
- Challenge problems
- Real-world applications

### 5. Key Takeaways
Summary of important points.

### 6. Next Steps
What to learn next.

---

## Teaching Principles

1. **Start from Basics**: Don't assume prior knowledge
2. **Build Intuition**: Explain why, not just how
3. **Use Analogies**: Connect to familiar concepts
4. **Encourage Practice**: Learning by doing
5. **Provide Feedback**: Help correct misunderstandings

---
